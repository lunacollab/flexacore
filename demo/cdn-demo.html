Usage Examples

Basic Setup

<!-- Include FlexaCore CSS --> <link rel="stylesheet" href="https://cdn.flexacore.dev/flexacore.min.css"> <!-- Include FlexaCore JavaScript --> <script src="https://cdn.flexacore.dev/flexacore-engine.min.js"></script> <script src="https://cdn.flexacore.dev/flexacore-framework.min.js"></script>

Initialize Framework

// Auto-initialize (default) // Framework will initialize automatically // Manual initialization FC.init({ theme: 'dark', debug: true, components: { modal: true, toast: true } });

Theme Management

// Apply theme FC.theme.apply('dark'); // Toggle theme FC.theme.toggle(); // Get current theme const currentTheme = FC.theme.get();

Component Usage

// Show modal FC.components.modal.show('my-modal'); // Show toast FC.components.toast.show('Hello World!', 'success', 3000); // Show tab FC.components.tabs.show('.tabs-container', 'tab2');
